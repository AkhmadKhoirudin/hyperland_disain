## ✨ Fitur Utama
- 🎨 **Desain Modern:** Perpaduan warna yang serasi dengan transparansi blur.
- 🚀 **One-Line Installer:** Instalasi otomatis tanpa ribet.
- 📊 **Waybar Estetis:** Status bar dengan indikator baterai, CPU, RAM, dan kalender.
- 🔍 **Wofi Launcher:** Menu aplikasi yang cepat dan minimalis.
- 🔔 **SwayNC:** Pusat notifikasi bergaya modern.
- 🛠️ **Auto Hardware Detect:** Mendeteksi GPU (Nvidia/AMD/Intel) secara otomatis saat instalasi.

## 🚀 Cara Instalasi (Sangat Cepat!)

## ⌨️ Shortcut Penting (Keybindings)
- `SUPER` + `Q` : Membuka Terminal (Kitty)
- `SUPER` + `E` : Membuka File Manager (Thunar)
- `SUPER` + `R` : Membuka App Launcher (Wofi)
- `SUPER` + `C` : Menutup Jendela
- `SUPER` + `M` : Keluar dari Hyprland (Wlogout)
- `SUPER` + `V` : Toggle Floating Mode
